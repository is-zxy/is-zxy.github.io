[国家集训队] 排队

发表于 2020-04-20 | 分类于 题解

排排坐，吃果果，生果甜嗦嗦，大家笑呵呵。你一个，我一个，大的分给你，小的留给我，吃完果果唱支歌，大家乐和和。

红星幼儿园的小朋友们排起了长长地队伍，准备吃果果。不过因为小朋友们的身高有所区别，排成的队伍高低错乱，极不美观。设第 $i$ 个小朋友的身高为 $h_i$。

幼儿园阿姨每次会选出两个小朋友，交换他们的位置，请你帮忙计算出每次交换后，序列的逆序对数。为方便幼儿园阿姨统计，在未进行任何交换操作时，你也应该输出该序列的逆序对数。

数据范围

对于 $15\%$ 的数据，$n,m \le 15$；
对于 $30\%$ 的数据，$n,m \le 200$；
另有 $15\%$ 的数据，$h_i$​ 各不相同；
另有 $15\%$ 的数据，$110 \le h_i \le 160$；
以上两类数据交集为空。

对于 $100\%$ 的数据，$1 \le m \le 2\times 10^3$，$1 \le n \le 2 \times 10^4$，$1 \le h_i \le 10^9$，$a_i \ne b_i$，$1 \le a_i,b_i \le n$。

思路

我喜欢暴力数据结构，所以我就用暴力分块解决了本题。

先归并排序求出原本序列的逆序对数。之后每次操作都在其基础上进行加减即可。

假设交换了$x$与$y$位置上的数$h[x]$与$h[y],(x< y)$，则在区间$(x,y)$内，一个数大于$h[x]$或小于$h[y]$则逆序对数增加一对，反之则减少一对。

据此，分块维护，二分查询块内与大于或小于$h[x]$以及$h[y]$的数的数量，更改当前序列内逆序对数即可。

应该都是分块的基本操作。
