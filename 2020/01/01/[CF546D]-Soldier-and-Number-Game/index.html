[CF546D] Soldier and Number Game

发表于 2020-01-01 | 分类于 题解

给定$n$，接下来有$n$组$a,b$，求$a!/b!$的质因子个数。

链接

CF546D Soldier and Number Game

思路

定义$sum[x]$表示$x$的质因子个数。

易得$sum[xy]=sum[x]+sum[y]$。

那么就可以用线性筛求出$1$到$5000000$的$sum$值，阶乘的话，根据上面的式子，$sum[!x]=\sum_{i=1}^xsum[i]$，于是求前缀和即可。

最后针对每组$a,b$，前缀和相减。
